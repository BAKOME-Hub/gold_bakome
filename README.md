## 🛠️ Installation

1. Download `gold_bakome.mq5` from this repository
2. Place the file in `MQL5/Experts/` (MT5) or `MQL4/Experts/` (MT4)
3. Compile (F7) in MetaEditor
4. Attach to a **XAUUSD M5 chart**
5. Configure risk settings (RiskPercent = 1% recommended)
